¿What's WebOffice?

It is a Firefox OS application that lets you read any file on OpenDocumentFormat (ODF) format stored in the SD memory your device. OpenDocument is an open standard format for storing office documents such as spreadsheets, text, graphics and presentations file.

Construction Criticadores Community Os Firefox

¿What kind of formats I can read?

Supported Documents are those with extensions, .Odt for text documents, .Ods for spreadsheets and .Odp for presentations

Construction Criticadores Community Os Firefox

¿How I can convert Microsoft Office documents in OpenDocument?

The application of Microsoft Office, has a conversion system and saved to convert any document either word, power point and excel in OpenDocument, soon in future updates will add a system for converting documents directly from WebOffice.

Construction Criticadores Community Os Firefox

¿The documents do not look like Microsoft Office?

There are some features of Microsoft Office, OpenDocument not supported therefore some things are not displayed correctly, for explanation see compatibilities: Microsoft.com
